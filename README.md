# Student Management System

## Task Overview

### Task 1: Project Setup
Created a Node.js project using `npm init -y`, installed Mongoose, and initialized the project structure.

### Task 2: MongoDB Connection
Connected the Node.js application to MongoDB using Mongoose and successfully created the `schoolDB` database.

### Task 3: Student Schema Creation
Designed a Student Schema with fields such as name, age, course, email, and marks, including required validations and default values.

### Task 4: Student Model Creation
Created a Student Model from the schema to perform database operations.

### Task 5: Insert Student Records
Inserted multiple student documents into the MongoDB database using Mongoose.

### Task 6: Read Operations
Retrieved all student records and fetched individual student records using specific conditions.

### Task 7: Query Operations
Performed various MongoDB queries such as filtering students by marks, course, age, and marks range.

### Task 8: MongoDB Concepts
Studied and understood the relationship between Database, Collection, Document, Schema, and Model using project examples.

### Task 9: Library Management System
Created a separate Book Schema and Model, inserted book records, and displayed all books from the database.
